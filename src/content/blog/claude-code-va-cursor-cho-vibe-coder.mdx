# Claude Code & Cursor: Cặp Bài Trùng Của Vibe Coder

Năm ngoái, tôi từng chật vật cả tuần chỉ để sửa một cái lỗi CSS lệch khung trong giao diện Mobile của một dự án Next.js. Mặc dù đã dùng ChatGPT để hỏi, nhưng tôi phải liên tục copy code từ editor sang trình duyệt, rồi lại copy câu trả lời của AI dán lại vào editor.

Đó là cách làm của thời "đồ đá". 

Sang năm 2026, [Vibe Coding](/blog/vibe-coding-huong-dan-toan-dien) lên ngôi nhờ sự xuất hiện của một cặp bài trùng huỷ diệt: **Cursor IDE** và **Claude 3.7 Sonnet**. Nếu bạn không biết code nhưng muốn tự tạo phần mềm, đây là 2 công cụ duy nhất bạn cần biết.

> **Tóm tắt nhanh:** Cursor là một trình soạn thảo code (như VS Code) nhưng được tích hợp sẵn AI. Claude 3.7 Sonnet là model AI viết code xuất sắc nhất hiện tại. Khi kết hợp chúng lại với tính năng "Composer" của Cursor, bạn không cần phải copy/paste code nữa. AI sẽ tự đọc toàn bộ dự án của bạn và viết code trực tiếp vào các file. 

## Cursor Không Phải Là Một IDE Bình Thường

Nhiều người nói: *"Tôi cài VS Code và dùng extension là được rồi, cần gì tải Cursor"*. Sai lầm!

Cursor là một nhánh tách ra (fork) từ VS Code, nhưng nó được xây dựng lại từ đầu để tối ưu cho AI. Tính năng đáng sợ nhất của nó là **Composer (Cmd/Ctrl + I)**.
Thay vì bạn hiện ra một cái khung chat để tán gẫu, Composer là một cửa sổ điều khiển. Bạn gõ: *"Tạo cho tôi một trang đăng nhập giống Apple.com"*. Cursor sẽ tự động tạo file `login.tsx`, tự động thêm css, tự động nối route, và nó chạy mượt mà ngay trước mắt bạn.

## Tại Sao Lại Là Claude 3.7 Sonnet?

Cursor cho phép bạn chọn nhiều model AI (GPT-4o, Gemini, Claude). Nhưng tại sao giới Vibe Coder lại tôn sùng Claude 3.7 Sonnet?

1. **Context Window 200K cực sâu:** Nó có thể đọc hiểu cùng lúc 50 file code trong dự án của bạn mà không bị "quên" (amnesia) như ChatGPT.
2. **Ít phá code cũ:** ChatGPT rất hay có tật "sửa lỗi A lòi ra lỗi B". Claude 3.7 lại cực kỳ cẩn thận. Nó biết cách chèn code mới vào mà không làm gãy cấu trúc cũ.
3. **Claude Code (CLI):** Gần đây Anthropic tung ra công cụ dùng trên terminal (dòng lệnh) tên là Claude Code. Nó có thể tự động dò lỗi (debug) và tự đưa ra quyết định mà không cần bạn phải duyệt từng dòng.

## Cuộc Suy Thoái Của Github Copilot

Github Copilot từng là vị vua năm 2023. Nhưng nó là một công cụ *Autocomplete* (tự động điền code). Tức là bạn vẫn phải biết gõ code, nó chỉ đoán xem bạn định gõ chữ gì tiếp theo.

Vibe Coder không làm thế. Chúng ta không tự gõ code. Chúng ta viết Prompt (ra lệnh bằng ngôn ngữ tự nhiên) để AI tự gõ. Đó là lý do Copilot trở nên yếu thế trước sức mạnh "Generate" (Tạo mới) của Cursor + Claude.

## Quy Trình Vibe Coding Với Cursor & Claude

Nếu bạn là người mới, đây là workflow chuẩn của tôi:

**Bước 1: Khởi tạo dự án**
Bấm `Cmd + I` (mở Composer trong Cursor). Nhập: *"Tạo một Next.js app mới với Tailwind CSS. Trang chủ là một Landing page bán khoá học Yoga"*. 
$\rightarrow$ Đứng dậy pha cafe. Cursor sẽ tự cào code và setup.

**Bước 2: Gắn rules (Luật lệ)**
Để Cursor không viết code rác, tôi đưa cho nó file `.cursorrules` (chứa các quy định về chuẩn code). Mỗi dự án tôi có một rule riêng.

**Bước 3: Tương tác (Vibe)**
Khi bị lỗi màn hình đỏ, tôi không cần biết lỗi đó là gì. Tôi bấm phím tắt `Cmd + L` (mở khung chat), quăng lỗi đó vào và bảo: *"Sửa giùm cái lỗi này"*. Claude trong Cursor sẽ tự lùng sục trong 10 file để tìm ra cái ngoặc bị thiếu.

## Checklist: Setup Môi Trường Vibe Coding

- [ ] **Tải Cursor IDE:** Miễn phí, có thể import mọi settings từ VS Code cũ của bạn sang chỉ với 1 click.
- [ ] **Cài đặt API Key hoặc Đăng ký Pro:** Bản free của Cursor khá giới hạn. Tôi khuyên bạn nên đăng ký bản Pro ($20/tháng) để dùng model xịn nhất.
- [ ] **Chuyển model sang `claude-3.7-sonnet`:** Vào Settings $\rightarrow$ Models và set Claude làm mặc định.
- [ ] **Tạo file `.cursorrules`:** Đặt ở thư mục gốc của dự án để ép AI viết code theo chuẩn.

## FAQ — Các Câu Hỏi Thường Gặp

**1. Cursor có an toàn bảo mật không?**
Cursor có chế độ "Privacy Mode" (chỉ dành cho tài khoản Pro), cam kết không lưu trữ code của bạn trên server của họ để train AI. Tuy nhiên, nếu bạn làm dự án ngân hàng tuyệt mật, hãy hỏi sếp trước khi dùng.

**2. Tôi mù tịt về code, dùng Cursor có làm được app không?**
Có, nhưng bạn cần học cách "Tư duy hệ thống" và cách "Viết prompt". Cursor là cái cưa máy, nhưng bạn vẫn phải biết cách đo và cắt gỗ.

**3. Có công cụ nào giống Cursor không?**
Gần đây có Windsurf (của Codeium) và Zed (tích hợp AI) cũng rất mạnh. Nhưng tính đến đầu 2026, Cursor vẫn đang vô đối về mặt trải nghiệm UX cho tính năng Composer.
